Software

Open-source tools developed for reaction discovery, atomistic simulation setup, phonon analysis, and synchrotron workflows.

reactit

Reaction Iterator

Generates all possible reactions between species efficiently.

GitHub repository

arcs

Automated Reactions for CO2 Storage

A package for studying impurity reactions in CO2 capture streams using stochastic sampling of DFT-based reaction networks.

Collaboration with Equinor, TotalEnergies, and Shell.

Original repository|Equinor fork

SMILESbox

Simulation Cells from SMILES Strings

Generates simple simulation boxes directly from organic SMILES strings.

GitHub repository

puppy

Phonon Unfolding and Projections

Unfolds defect phonons and projects selected eigendisplacements onto phonon dispersion curves.

GitHub repository

beavermap

Beam Weaver Mapping

Multiprocessing, analysis, and plotting utilities for ESRF synchrotron data.

GitHub repository
